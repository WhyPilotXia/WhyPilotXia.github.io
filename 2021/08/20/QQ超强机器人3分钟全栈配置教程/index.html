QQ超强机器人3分钟全栈配置教程

发表于2021-08-20|更新于2021-08-20
|字数总计:288|阅读时长:1分钟|阅读量:

话不多说，先上教程，3分钟实现从零到机器人启动成功。

从零指连python环境都没有。

1.微软应用商店搜索python3.9安装

2.打开cmd运行一句话命令：

pip install pydantic loguru pygtrie httpx python-dotenv uvicorn fastapi requests selenium aiocqhttp ffmpeg aiohttp nb-cli nonebot2 nonebot-plugin-apscheduler nonebot-adapter-cqhttp -i https://pypi.tuna.tsinghua.edu.cn/simple

如果下载慢，改成分三次执行：

1.pip install pydantic loguru pygtrie httpx python-dotenv uvicorn fastapi requests selenium

2.pip install aiocqhttp ffmpeg aiohttp nb-cli

3.pip install nonebot2 nonebot-plugin-apscheduler nonebot-adapter-cqhttp

3.浏览器访问[直链下载地址](https://codeload.github.com/89461561511656/nonebot2/zip/refs/heads/main

)

如果不行，可以在QQ群‘全栈’中下载文件

4.解压后

在这个位置创建.\nonebot2-main\zhuzhu\go-cqhttp-v0.9.40-fix4-windows-amd64\1.cmd，内容是同目录下那个exe的绝对路径，如C:\Users\zyh\Downloads\Compressed\nonebot2-main\zhuzhu\go-cqhttp-v0.9.40-fix4-windows-amd64\go-cqhttp_windows.exe

保存后运行该文件，并运行上级目录的bot.py，按要求QQ扫码（一般是小号）即可使用。

5.享受成功的喜悦！

给机器人私聊发送/echo hello world!

更多功能懂得都懂
